# 理财收益计算器 UI 说明 (Financial Calculator UI Specification)

## 界面布局 (Interface Layout)

### 顶部导航栏 (App Bar)
- 标题：理财收益计算器
- Material Design 3 风格
- 带有阴影效果

### 主要内容区域 (Main Content Area)

#### 1. 本金输入卡片
- 图标：💰 钱包图标 (account_balance_wallet)
- 标题："本金金额 (CNY)"
- 输入框：
  - 提示文本："请输入本金金额"
  - 数字键盘，支持小数点
  - 默认值：10000
  - 浅灰色背景 (#F5F5F5)
  - 圆角边框 (8px)

#### 2. 人民币利率卡片
- 图标：📈 趋势向上图标 (trending_up)
- 标题："人民币理财年化利率 (%)"
- 输入框：
  - 提示文本："例如: 3.5"
  - 数字键盘，支持小数点（最多2位）
  - 实时计算
  - 圆角边框和浅色背景

#### 3. 美元利率卡片
- 图标：📈 趋势向上图标 (trending_up)
- 标题："美元理财年化利率 (%)"
- 输入框：
  - 提示文本："例如: 4.2"
  - 数字键盘，支持小数点（最多2位）
  - 实时计算
  - 圆角边框和浅色背景

#### 4. 汇率输入卡片
- 图标：💱 货币兑换图标 (currency_exchange)
- 标题："人民币/美元汇率"
- 输入框：
  - 提示文本："例如: 7.2"
  - 默认值：7.2
  - 数字键盘，支持小数点
  - 圆角边框和浅色背景

#### 5. 日期选择卡片
- 图标：📅 日历图标 (calendar_today)
- 标题："理财到期日"
- 日期选择器：
  - 显示格式：YYYY-MM-DD
  - 点击打开日期选择对话框
  - 默认文本："请选择到期日期"（灰色）
  - 选择后显示日期（黑色）
  - 下拉箭头图标
  - 圆角边框和浅色背景

#### 6. 计算按钮
- 图标：🧮 计算器图标 (calculate)
- 文本："计算收益"
- 样式：
  - 大字体 (18px)
  - 充满宽度
  - 圆角 (12px)
  - 主题色背景
  - 白色文字
  - 较大内边距 (16px 垂直)

#### 7. 结果显示区域

##### 未计算状态
- 灰色提示卡片
- 文本："请输入完整信息后点击计算"
- 居中显示

##### 已计算状态

- 加粗蓝色文字
- 居中对齐

**人民币收益卡片**

- 左上角：
  - 绿色圆角方框

  - 标题："人民币理财收益" (18px, 加粗)
- 中央：
  - 超大字体 (32px)
  - 加粗

**美元收益卡片**

  - 标题："美元理财收益" (18px, 加粗)

  - 超大字体 (32px)
  - 加粗

## 交互特性 (Interaction Features)

### 实时计算
- 所有输入框的值改变时，会触发重新计算
- 使用 GetX 响应式编程，UI 自动更新

### 输入验证
- 数字输入框限制只能输入数字和小数点
- 小数点后最多保留2位
- 负数和无效输入会被处理为0

### 日期选择
- 只能选择今天或未来的日期
- 使用系统原生的日期选择器
- 选择后自动格式化显示

### 视觉反馈
- 所有卡片有轻微阴影（elevation: 2）
- 按钮有按压效果
- 输入框获得焦点时有高亮边框

## 响应式设计 (Responsive Design)

- 所有组件充满屏幕宽度
- 适当的间距（16px）
- 可滚动内容区域
- 适配不同屏幕尺寸

## 主题支持 (Theme Support)

- 支持亮色主题和暗色主题
- 使用 Material Design 3 配色方案
- 主色调：蓝色 (Colors.blue)
- 自动适配系统主题设置

## 颜色方案 (Color Scheme)

### 亮色模式
- 背景：白色
- 卡片：白色（阴影）
- 输入框背景：浅灰色 (#F5F5F5)
- 主色：蓝色
- 人民币：绿色系
- 美元：橙色系

### 暗色模式
- 背景：深色
- 卡片：深灰色（阴影）
- 输入框背景：更深的灰色
- 主色：浅蓝色
- 保持良好的对比度

## 用户体验亮点 (UX Highlights)

1. **默认值预填充**: 汇率和本金有合理的默认值，方便快速测试
2. **清晰的视觉层次**: 使用卡片、图标和颜色区分不同功能区
3. **即时反馈**: 输入后立即可以看到计算结果
4. **错误处理**: 输入无效时不会崩溃，而是显示0或提示
5. **国际化友好**: 使用图标和清晰的标签，易于理解
6. **可访问性**: 使用充足的色彩对比度和可读的字体大小
