## Color Scheme Details

### Light Mode
- **App Bar**: Blue (#2196F3 or similar Material Blue)
- **Background**: White (#FFFFFF)
- **Cards**: White with subtle shadow (elevation: 2)
- **Input Fields**: Light gray background (#F5F5F5)
- **Duration Card**: Light blue background (#E3F2FD)
- **RMB Result Card**: Light green background (#E8F5E9)
- **USD Result Card**: Light orange background (#FFF3E0)

### Dark Mode
- **App Bar**: Dark blue
- **Background**: Dark gray (#121212)
- **Cards**: Dark gray with elevation
- **Input Fields**: Darker gray
- **Duration Card**: Dark blue tint
- **RMB Result Card**: Dark green tint
- **USD Result Card**: Dark orange tint

## Key UI/UX Features

1. **Card-based Design**: Each input section is contained in a rounded card with shadow
2. **Icon Indicators**: Each field has a relevant icon for quick recognition
3. **Real-time Validation**: Input fields only accept numeric values
4. **Responsive Layout**: Full-width cards with appropriate spacing
5. **Visual Hierarchy**: Results are prominently displayed with large text
6. **Color Coding**: Different colors for different currencies (green for RMB, orange for USD)
7. **Professional Look**: Material Design 3 principles throughout
8. **Accessibility**: High contrast, readable fonts, clear labels

## Responsive Behavior

- All content is scrollable on smaller screens
- Cards maintain 16px padding on all sides
- Input fields expand to full width within cards
- Button is full-width for easy tapping
- Results scale proportionally
